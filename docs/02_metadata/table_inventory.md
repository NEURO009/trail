# Table Inventory — IRIS Platform

This document catalogs all physical tables, database schemas, and storage files read and written by the IRIS pipeline.

---

## 1. Table Directory

| Table Identity | Format | Storage Zone | Primary CRUD Operations | Logical Role & Description |
|---|---|---|---|---|
| `FACT_POS_SYNDICATED` | Delta / Snowflake | Internal Source DW | Read | Raw POS syndicated scanner data from stores and markets. |
| `DIM_PROMOTION` | Delta / Snowflake | Internal Source DW | Read | Retail promotion lookup table mapping promotion keys to labels (e.g. Feat, Disp). |
| `DIM_PRODUCT` | Delta / Snowflake | Internal Source DW | Read | SKU mapping hierarchy. Maps product keys to brands and categories. |
| `DIM_MARKET` | Delta / Snowflake | Internal Source DW | Read | Geography lookup mapping retailer market identifiers to regions. |
| `DIM_TIME` | Delta / Snowflake | Internal Source DW | Read | Fiscal 4-5-4 retail calendar calendar details. |
| `DIM_UPC_PRODUCT` | Parquet / Delta | ADLS Gen2 | Read | Detailed physical attributes of UPC products (roll count, ply size, sheets). |
| `NIELSEN_METRICS` | Parquet | ADLS Gen2 | Read / Write | Cleaned syndicated POS data with parsed promotion variables. |
| `AMAZON_METRICS` | Parquet | ADLS Gen2 | Read / Write | Formatted Amazon sales records aligned to the standard POS schema. |
| `IRIS_FACTS_BASE_METRICS` | Parquet | ADLS Gen2 | Read / Write | Master UPC facts enriched with category configs, search rank, and availability. Replaced low volume UPC PPGs with subtotal aggregates. |
| `IRIS_FACTS_SUB_METRICS` | Parquet | ADLS Gen2 | Write | Keeps record of raw, unmodified subtotal records. |
| `IRIS_FACTS_AGG_METRICS` | Parquet | ADLS Gen2 | Read / Write | PPG, Brand, and Category rollups of metrics with stick algorithm price smoothing. |
| `IRIS_REF_PRODUCT_DISTANCE` | Parquet | ADLS Gen2 | Read / Write | Pairs UPC codes with similarity distances derived from Circana models. |
| `IRIS_REF_COMP_PRODUCT_MAPPING`| Parquet | ADLS Gen2 | Read / Write | Stores competitor set assignments per market. |
| `IRIS_REF_COMP_INTERACTION_WEIGHTS` | Parquet | ADLS Gen2 | Read / Write | Competitive interaction weight references partitioned by category. |
| `IRIS_FACTS_COMP_INTERACTION_METRICS` | Parquet | ADLS Gen2 | Read / Write | Competitor metrics weighted by interaction weights. |
| `IRIS_FACTS_SEASONALITY` | Parquet | ADLS Gen2 | Read / Write | Weekly seasonality multipliers generated by distributed Prophet runs. |
| `COEF_HISTORY` | Parquet | ADLS Gen2 | Read / Append | Archive of raw modeling coefficients across run history. |
| `IRIS_MODEL_COEF_CURRENT` | Parquet | ADLS Gen2 | Read / Write | Current deployed modeling coefficients. Outliers are replaced and missing scopes backfilled. |
| `MODEL_DECOMP` | Parquet | ADLS Gen2 | Read / Write | Attributed volumes mapping coefficients to feature changes. |
| `CATEGORY_INCREMENTALITY` | Parquet | ADLS Gen2 | Read / Write | Decompositions aggregated to incrementality and cannibalization rates. |
| `PROACTIVE_ALERTS_FACTS` | Delta / Parquet | ADLS Gen2 | Read / Write | Generated alert anomalies across base price, promo frequency, ACV distribution, etc. |
| `SCENARIO_MODEL_COEFFICIENTS` | SQL Table | Azure SQL Server | Truncate / Write | Blended pricing scenario coefficients serving the front-end simulation dashboard. |
| `SCENARIO_HISTORICAL_FACTS` | SQL Table | Azure SQL Server | Truncate / Write | Historical POS facts serving front-end charts. |
| `PEA_DATA_TABLES` | SQL Table | Azure SQL Server | Read / Write | Trade promotion event outputs protected by SQL Row-Level Security. |
| `SNOWFLAKE_REPORTING_MART` | Snowflake views | Snowflake Secure | Read / Write | Consolidated analytics database serving Power BI dashboards. |
